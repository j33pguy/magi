# Roadmap

Status: Draft

MAGI is model- and agent-agnostic, self-hosted, and has zero cloud dependency by default.

This roadmap is planning-only and not a commitment. It will change as implementation and feedback evolve.

## Near-Term Themes

- stabilize single-node performance and correctness
- improve access control and identity handling
- expand sync coverage and adapter support
- make onboarding and deployment smoother

## Longer-Term Themes

- role-separated scale-out deployments
- richer project rehydration flows
- stronger policy and audit capabilities
